A simple Minecraft Forge client mod that shortens the tooltip of trimmed armor. Idea comes from [this project](https://github.com/Andrew6rant/Better-Trim-Tooltips), but doesn't include the compat with StackedTrims. The brackets around the trim material aren't hardcoded, so they can be changed in lang files. For default, en_us uses half-width () and zh_cn uses full-width （）.

***
一个简单的Minecraft Forge客户端模组，减短了盔甲纹饰工具提示。灵感来自[此项目](https://github.com/Andrew6rant/Better-Trim-Tooltips)，但是不包含和StackedTrims的联动。纹饰材料两侧的括号非硬编码，可以在语言文件中修改。默认情况下，en_us使用半角()，zh_cn使用全角（）。
